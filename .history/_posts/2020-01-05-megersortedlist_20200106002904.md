将两个有序链表合并为一个新的有序链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。 

示例：

输入：1->2->4, 1->3->4
输出：1->1->2->3->4->4

来源：力扣（LeetCode）

### 题解
1.非递归  
``` java
 public ListNode mergeTwoLists(ListNode l1, ListNode l2) {
       ListNode dummy = new ListNode(0);
       ListNode p = dummy;
       while (l1 != null && l2 != null) {
           if (l1.val <= l2.val) {
               p.next = l1;
               l1 = l1.next;
           } else {
               p.next = l2;
               l2 = l2.next;
           }
           p = p.next;
       }
       p.next = l1 == null ? l2 : l1;
       return dummy.next; 
    }
```  
2.递归  
``` java
public ListNode mergeTwoLists(ListNode l1, ListNode l2) {
    // termination
        if(l1 == null) {
            return l2;
        }
        if(l2 == null) {
            return l1;
        }
        //process
        if(l1.val < l2.val) {
            // drill down
            l1.next = mergeTwoLists(l1.next, l2);
            return l1;
        } else {
            l2.next = mergeTwoLists(l1, l2.next);
            return l2;
        }
    }
```
